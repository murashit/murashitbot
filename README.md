# Murashitbot

a Marcov-Chain Twitter Bot like @murashittest

## Requirements
* `ruby 1.9`
* `Mecab`
* `macab-ruby`

## Usage

### 1. Initialize

    $ mkdir bots; cd bots
    $ murashitbot init USERNAME

### 2. Edit USERNAME/source.txt

### 3. Parse

    $ murashitbot parse USERNAME

### 4. Tweet!!

    $ murashitbot tweet USERNAME
    $ murashitbot reply USERNAME
